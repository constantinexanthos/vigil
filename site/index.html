// how it works

30 seconds to full visibility

No config files. No API keys. No cloud accounts. Install and go.

1
brew install vigil
Single binary. 5MB. Zero dependencies.
2
vigil watch ~/projects
Daemon starts. Watches filesystem, git, and processes.
3
Start working.
Launch Claude Code, open Cursor, spin up Conductor. Everything captured.
4
vigil status
Active agents, files touched, collision warnings, burn rate. Menu bar always visible.

// boundaries

What Vigil is not

Not a code review tool
CodeRabbit reviews PRs after they're opened. We operate during execution, before the PR exists.
Not an orchestrator
Conductor spawns agents. We observe and verify them regardless of how they were launched.
Not LLMOps
Langfuse tracks API calls. We track code actions: files modified, imports resolved, architecture drift.
Not an AI assistant
We don't write code. We watch the things that write code.
